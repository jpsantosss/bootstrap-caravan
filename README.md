# Caravan

A modern, fully responsive web project built primarily with the **Bootstrap** framework. This repository was created as part of a Bootstrap course to deeply understand and apply its powerful grid system, responsive design principles, and pre-built UI components.

## 🚀 Live Access

Check out the live application here: **[Caravan - Live Demo](https://jpsantosss.github.io/bootstrap-caravan/)**

## 🛠️ Technologies

- **HTML5** - Semantic markup
- **CSS3** - Custom styling and layout adjustments
- **Bootstrap** - Front-end framework for responsive, mobile-first development

## ✨ Key Features

- **Mobile-First Design:** Fully responsive layout that adapts seamlessly to all screen sizes (mobile, tablet, and desktop).
- **Bootstrap Grid System:** Efficient use of rows and columns for structuring the page layout.
- **UI Components:** Implementation of ready-to-use Bootstrap components such as Navbars, Cards, Modals, Buttons, and Carousels.
- **Clean Interface:** A professional and polished user interface designed for a great user experience.

## 💻 Getting Started

To run this project locally, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/jpsantosss/bootstrap-caravan.git

2. **Navigate to the project directory:**
   ```bash
   cd bootstrap-caravan
   ```

3. **Run the application:**
   Simply open the index.html file in your preferred web browser.
